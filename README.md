# Plataforma de Servicios Freelance

Proyecto desarrollado con Symfony 7 para conectar freelancers con clientes.

## 🚀 Tecnologías
- Symfony 7
- PHP 8.3+
- Doctrine ORM
- JWT Auth
- Docker (para despliegue final)

## 🧪 Requisitos
- PHP 8.3
- Composer
- Docker & Docker Compose
- Git

## 🛠 Instalación
```bash
composer install
cp .env .env.local
# configura tu base de datos en .env.local
